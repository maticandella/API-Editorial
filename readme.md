# API Editoriales

Esta API permite gestionar las operaciones del lado del servidor para una Editorial de Libros.

## Proposito 🚀

En este proyecto iré volcando todos los conocimientos adquiridos en torno a Node.js.

## Construido con 🛠️

* Node.js
* Express
* Express-validator
* Mssql
* Dotenv
* Cors
